**Ordered list**
```
1. First Item
2. Next item
3. Another item
   1. Sub item
   2. Another sub item
4. Etc.
```


**Unordered list**
```
* Next item
* Nother item
* Etc.
```

Colons can be used to align columns.
```
| Tables        | Are           | Cool  |
| ------------- |:-------------:| -----:|
| col 3 is      | right-aligned | $1600 |
| col 2 is      | centered      |   $12 |
| zebra stripes | are neat      |    $1 |
```
